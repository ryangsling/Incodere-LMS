# ILMS - Incodere Learning Management System

## Prerequisites

- Node.js 22+
- npm 10+

## Setup

### Backend

```bash
cd backend
cp ../.env .env
npm install
npm run dev
```

### Frontend

```bash
cd frontend
cp ../.env .env
npm install
npm run dev
```

## Environment Variables

See `.env.example` for all required variables.
